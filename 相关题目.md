以下代码中哪个是类、对象、局部变量、成员变量、类变量/静态变量？这几个变量分别初始化了多少次？
```java
public class C {
    public static final boolean constant = true;
    private int member;

    public C(int member) {
        this.member = member;
    }

    public void fun() {
        int memberTimes2 = member * 2;
        System.out.println(memberTimes2);
    }

    public static void main(String[] args) {
        C[] cArray = new C[10];
        for (int i = 0; i < 10; i++) 
            cArray[i] = new C(i);
        for (int i = 0; i < 20; i++)
            cArray[i % 10].fun();
    }
}
```
